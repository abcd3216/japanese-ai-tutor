# 文法：〜てしまう（N4）
用法：① 不小心、無意間做了某事（帶有後悔或遺憾語氣）；② 動作已完全結束（強調完成）
結構：動詞て形 + しまう（口語縮略：〜ちゃう／〜じゃう）
關鍵字：てしまう, ちゃう, じゃう, 不小心, 後悔, 遺憾, 完成, te shimau, 意外
例句：
  財布を忘れてしまった。（不小心把錢包忘了。）
  ケーキを全部食べてしまった。（把蛋糕全吃完了。）
  宿題をなくしてしまいました。（不小心把作業弄丟了。）
注意：口語常縮成「〜ちゃった」（食べちゃった）；強調「完成」時無後悔語氣，依語境判斷
來源：japanesetest4you.com — JLPT N4 Grammar〜てしまう；みんなの日本語 初級II 第24課（スリーエーネットワーク）

# 文法：〜ておく（N4）
用法：事先做好某件事以備後用，強調「預先準備」
結構：動詞て形 + おく（口語縮略：〜とく）
關鍵字：ておく, とく, 事先, 預先準備, 提前, te oku, 備用
例句：
  旅行の前にホテルを予約しておきます。（旅行前先預訂好飯店。）
  会議の資料を準備しておいた。（事先準備好了會議資料。）
  冷蔵庫に食べ物を入れておいてください。（請先把食物放進冰箱。）
注意：口語縮成「〜とく」（予約しとく）；強調行動是為了未來某個目的而提前做
來源：japanesetest4you.com — JLPT N4 Grammar〜ておく；みんなの日本語 初級II 第24課（スリーエーネットワーク）

# 文法：〜てみる（N4）
用法：試著做某件事，表示嘗試、試看看
結構：動詞て形 + みる（禮貌體：〜てみます）
關鍵字：てみる, 嘗試, 試試看, 試著, te miru, 挑戰, 試驗
例句：
  この料理を作ってみました。（試著做了這道料理。）
  新しいカフェに行ってみよう。（試著去新開的咖啡廳吧。）
  日本語で話してみてください。（請試著用日文說說看。）
注意：「みる」來自動詞「見る」（看），引申為「嘗試看看」；結果好壞不確定，強調嘗試的過程
來源：japanesetest4you.com — JLPT N4 Grammar〜てみる；Tae Kim's Guide to Japanese Grammar (guidetojapanese.org)

# 文法：〜てあげる／くれる／もらう（N4）
用法：描述「給予」與「接受」的動作，說明恩惠的方向
結構：動詞て形 + あげる（我給別人）／くれる（別人給我）／もらう（我從別人那裡獲得）
關鍵字：てあげる, てくれる, てもらう, 給予, 接受, 恩惠, ageru, kureru, morau, 授受動詞
例句：
  友達に本を貸してあげた。（把書借給了朋友。）[我→他人]
  先生が漢字を教えてくれた。（老師教了我漢字。）[他人→我]
  母に料理を作ってもらいました。（請媽媽幫我做了料理。）[我從他人獲得]
注意：「あげる」說話者為恩惠提供者；「くれる」說話者（或其群體）為恩惠接受者；「もらう」強調主動請求
來源：japanesetest4you.com — JLPT N4 Grammar てあげる・てくれる・てもらう；みんなの日本語 初級II 第24課（スリーエーネットワーク）

# 文法：〜ば〜のに（N4）
用法：表示「如果...就好了」，對現實相反的情況感到遺憾或惋惜（反事實假設）
結構：動詞／形容詞ば形 + のに（句尾帶遺憾語氣）
關鍵字：ばのに, ばよかった, 如果, 就好了, 遺憾, 惋惜, 反事實, 假設, 後悔
例句：
  もっと早く起きればよかったのに。（如果早點起床就好了。）
  お金があれば旅行できるのに。（如果有錢的話就能去旅行了。）
  もっと勉強すれば合格したのに。（如果再多讀一點就能合格了。）
注意：「のに」句尾帶有惋惜、遺憾語氣；「ば」條件是與現實相反的假設，不是真實條件
來源：japanesetest4you.com — JLPT N4 Grammar〜ばよかった；Tae Kim's Guide to Japanese Grammar (guidetojapanese.org)

# 文法：〜のに（逆接）（N4）
用法：表示「明明...卻...」，前後形成出乎意料的對比或抱怨
結構：普通體（現在形）+ のに + 後句（結果出乎意料）
關鍵字：のに, 逆接, 明明, 卻, 對比, 抱怨, 不滿, 出乎意料, noni
例句：
  一生懸命勉強したのに、試験に落ちた。（明明拼命讀書，卻考試落榜了。）
  薬を飲んだのに、まだ頭が痛い。（明明吃了藥，頭還是在痛。）
  約束したのに、来なかった。（明明約好了，卻沒來。）
注意：此「のに」是逆接接續詞，不同於表示目的的「〜のに（為了）」；帶有說話者的不滿或驚訝
來源：japanesetest4you.com — JLPT N4 Grammar〜のに（逆接）；みんなの日本語 初級II 第28課（スリーエーネットワーク）

# 文法：〜ようにする（N4）
用法：表示「努力讓自己做到...」，強調持續的意志或習慣養成
結構：動詞辭書形／ない形 + ようにする
關鍵字：ようにする, 努力, 習慣, 意志, 刻意, you ni suru, 養成
例句：
  毎日運動するようにしています。（我努力讓自己每天運動。）
  遅刻しないようにします。（我會努力不遲到。）
  野菜をたくさん食べるようにしてください。（請努力多吃蔬菜。）
注意：與「〜ようになる」比較：「ようにする」是主動努力（自己控制）；「ようになる」是自然變化（結果）
來源：japanesetest4you.com — JLPT N4 Grammar〜ようにする；みんなの日本語 初級II 第36課（スリーエーネットワーク）

# 文法：〜ようになる（N4）
用法：表示「（經過變化後）變得會...／能夠...」，描述能力或習慣的自然轉變
結構：動詞辭書形／ない形 + ようになる
關鍵字：ようになる, 變得, 能夠, 進步, 改變, you ni naru, 能力變化, 從不會到會
例句：
  練習したら、泳げるようになりました。（練習之後，變得會游泳了。）
  日本に来てから、箸が使えるようになった。（來日本之後，學會用筷子了。）
  薬を飲んで、よく眠れるようになりました。（吃了藥之後，變得能好好睡覺了。）
注意：強調「從不會到會」的過程變化；與「〜ようにする」（主動努力）對比，「〜ようになる」是結果
來源：japanesetest4you.com — JLPT N4 Grammar〜ようになる；みんなの日本語 初級II 第36課（スリーエーネットワーク）

# 文法：〜はずだ（N4）
用法：表示「按理說應該是...」，根據某種根據或邏輯推測出的結論
結構：動詞普通體 / 名詞 + のはずだ / な形容詞 + なはずだ
關鍵字：はずだ, 應該, 推測, 理應, hazu da, 邏輯推斷, 按理說
例句：
  彼はもう家に着いているはずです。（他按理說應該已經到家了。）
  この店は安いはずだ。（這家店應該很便宜。）
  田中さんは今日来るはずだったのに、来なかった。（田中應該今天會來的，卻沒來。）
注意：「はずだ」基於客觀根據推測；與「〜だろう」（主觀猜測）不同；否定「〜はずがない」= 不可能是
來源：japanesetest4you.com — JLPT N4 Grammar〜はずだ；Tae Kim's Guide to Japanese Grammar (guidetojapanese.org)

# 文法：〜らしい（N4）
用法：① 根據外部資訊推測「聽說／好像是...」；② 表示「很有...的樣子」（符合該事物特徵）
結構：普通體 + らしい（推測）；名詞 + らしい（典型特徵）
關鍵字：らしい, 聽說, 好像, 推測, 間接, rashii, 傳聞, 典型特徵
例句：
  明日は雨が降るらしい。（聽說明天會下雨。）[外部資訊推測]
  彼女はもう帰ったらしい。（她好像已經回去了。）[推測]
  彼はいかにも学生らしい。（他看起來很有學生的樣子。）[典型特徵]
注意：「らしい」來自外部資訊（聽說、看到）；與「〜そうだ」比較：「そうだ」是直接看起來的樣子，「らしい」是間接資訊
來源：japanesetest4you.com — JLPT N4 Grammar〜らしい；みんなの日本語 初級II 第32課（スリーエーネットワーク）
