# 文法：〜です（N5）
用法：表示「是...」，用於禮貌體句尾，說明身份或事物性質
結構：名詞 / な形容詞 + です
關鍵字：です, 是, 禮貌體, 肯定, 身份, 名詞句, desu
例句：
  私は学生です。（我是學生。）
  これは本です。（這是書。）
  田中さんは先生です。（田中先生是老師。）
注意：疑問句在句尾加「か」→ 学生ですか？；否定形為「〜ではありません」
來源：みんなの日本語 初級I 第1課（スリーエーネットワーク）；JLPT N5 文法範圍（国際交流基金）

# 文法：〜ます（N5）
用法：動詞禮貌體，表示客氣、正式的動作描述
結構：動詞ます形（第一類：-u → -i + ます；第二類：去る + ます；第三類：します、きます）
關鍵字：ます, 動詞禮貌體, 敬體, masu, 現在式, 未來式
例句：
  毎日日本語を勉強します。（我每天讀日文。）
  明日、学校に行きます。（明天去學校。）
  夜ご飯を食べます。（吃晚餐。）
注意：否定形為「〜ません」；過去形為「〜ました」；過去否定為「〜ませんでした」
來源：みんなの日本語 初級I 第3課（スリーエーネットワーク）；JLPT N5 文法範圍（国際交流基金）

# 文法：〜ない（否定形）（N5）
用法：表示「不做...」，動詞的否定形（普通體）
結構：第一類動詞：-u → -a + ない（書く→書かない）；第二類：去る + ない（食べ→食べない）；第三類：しない、こない
關鍵字：ない, 否定, 不做, 普通體, 動詞否定, nai
例句：
  今日は学校に行かない。（今天不去學校。）
  お肉を食べない。（不吃肉。）
  宿題をしない。（不做功課。）
注意：「ある」的否定為「ない」（不是あらない）；禮貌體否定為「〜ません」
來源：みんなの日本語 初級I 第17課（スリーエーネットワーク）；新完全マスターN5 文法（3A Corporation）

# 文法：〜てください（N5）
用法：禮貌地請求對方做某件事，等同「請...」
結構：動詞て形 + ください
關鍵字：てください, 請, 請求, 命令, te kudasai, 指示
例句：
  ここに名前を書いてください。（請在這裡寫名字。）
  静かにしてください。（請保持安靜。）
  窓を開けてください。（請打開窗戶。）
注意：更禮貌的說法為「〜ていただけますか」；禁止為「〜ないでください」（請不要...）
來源：みんなの日本語 初級I 第14課（スリーエーネットワーク）；JLPT N5 文法範圍（国際交流基金）

# 文法：〜たい（N5）
用法：表示說話者的「想要做...」的願望（第一人稱）
結構：動詞ます形去ます + たい（行きたい、食べたい）
關鍵字：たい, 想要, 願望, 希望, 想做, tai, 第一人稱
例句：
  日本へ行きたいです。（想去日本。）
  ラーメンを食べたい。（想吃拉麵。）
  日本語を上手になりたいです。（想把日文學好。）
注意：詢問他人用「〜たいですか」；否定為「〜たくない」；接助詞時「を」可換成「が」
來源：みんなの日本語 初級I 第13課（スリーエーネットワーク）；JLPT N5 文法範圍（国際交流基金）

# 文法：〜ている（N5）
用法：① 正在進行的動作（動作進行中）；② 狀態的持續（某動作完成後的狀態）
結構：動詞て形 + いる（禮貌體：〜ています）
關鍵字：ている, ています, 進行, 正在, 狀態, 持續, te iru, 現在進行式, 動作持續
例句：
  今、ご飯を食べています。（現在正在吃飯。）[進行]
  窓が開いています。（窗戶開著。）[狀態]
  田中さんは結婚しています。（田中先生已婚。）[狀態]
注意：區分「進行」與「狀態」很重要；「知っている」=「知道」，不用「知る」的現在式
來源：みんなの日本語 初級I 第14課（スリーエーネットワーク）；新完全マスターN5 文法（3A Corporation）

# 文法：〜に行く（N5）
用法：表示「去（某地）做某事」，目的地 + 行動目的
結構：動詞ます形去ます + に行く（食べに行く）；或 名詞 + を + しに行く
關鍵字：に行く, 去做, 目的, 外出, ni iku, 移動目的
例句：
  映画を見に行きます。（去看電影。）
  買い物に行きます。（去購物。）
  友達に会いに行った。（去見了朋友。）
注意：「に」表示目的；可換成「来る（来ます）」→ 〜に来る（來做某事）
來源：みんなの日本語 初級I 第13課（スリーエーネットワーク）；JLPT N5 文法範圍（国際交流基金）

# 文法：〜から（因為）（N5）
用法：表示原因或理由，說明「因為...所以...」
結構：普通體 / ます形 + から；句型：〜から、〜（原因在前）
關鍵字：から, 因為, 原因, 理由, kara, 因果關係, 所以
例句：
  眠いから、寝ます。（因為很睏，所以去睡。）
  雨だから、外に出ない。（因為下雨，不出去。）
  好きだから、毎日練習します。（因為喜歡，所以每天練習。）
注意：與「〜ので」比較：「から」主觀、直接；「ので」較客觀、委婉；後者更常用於正式場合
來源：みんなの日本語 初級I 第16課（スリーエーネットワーク）；新完全マスターN5 文法（3A Corporation）

# 文法：〜が好きです（N5）
用法：表示「喜歡...」，說明喜好的對象
結構：名詞 + が好きです（喜歡）；名詞 + が嫌いです（討厭）；名詞 + が得意です（擅長）
關鍵字：好きです, 喜歡, 好き, 嫌い, 討厭, suki, 喜好, 興趣
例句：
  音楽が好きです。（我喜歡音樂。）
  猫が大好きです。（我非常喜歡貓。）
  辛い食べ物が嫌いです。（我討厭辣的食物。）
注意：「好き」是な形容詞，不是動詞；助詞用「が」而非「を」；程度加「大」→「大好き」
來源：みんなの日本語 初級I 第12課（スリーエーネットワーク）；JLPT N5 文法範圍（国際交流基金）

# 文法：〜より〜のほうが（N5）
用法：比較兩者，表示「比起A，B更...」
結構：AよりBのほうが + 形容詞；疑問：AとBとどちら（どっち）が〜ですか
關鍵字：より, のほうが, 比較, 比起, yori, 更, A比B
例句：
  犬より猫のほうが好きです。（比起狗，更喜歡貓。）
  東京より大阪のほうが食べ物がおいしい。（比起東京，大阪的食物更好吃。）
  バスより電車のほうが速いです。（比起公車，電車更快。）
注意：「より」後面是比較基準（劣者）；「のほうが」後面是結論（優者）
來源：みんなの日本語 初級I 第12課（スリーエーネットワーク）；新完全マスターN5 文法（3A Corporation）

# 文法：〜も（也）（N5）
用法：表示「也」，追加相同的說明到另一個主語或受詞
結構：名詞 + も（取代「は」「が」「を」等助詞）
關鍵字：も, 也, 追加, mo, 同樣, 兩者都, 包含
例句：
  私は学生です。田中さんも学生です。（我是學生。田中也是學生。）
  りんごが好きです。みかんも好きです。（喜歡蘋果，也喜歡橘子。）
  日本語も英語も話せます。（日文和英文都會說。）
注意：否定句「〜も〜ない」= 兩者都不；「どこにも行かない」= 哪裡也不去
來源：みんなの日本語 初級I 第11課（スリーエーネットワーク）；JLPT N5 文法範圍（国際交流基金）
